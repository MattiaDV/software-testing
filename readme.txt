Librerie con comandi da installare per far funzionare il software:

pip install selenium
pip install undetected-chromedriver
pip install csv
pip install webdriver-manager
pip install setuptools
pip install --upgrade undetected-chromedriver
pip install webdriver_manager

P.S. se non funzionano con pip, utilizzare la vostra versione del pip (es: pip, pip2, pip3)
Se ci sono problemi richiedete assistenza al vostro superiore

(Per avviare il software, nel cmd o nel terminale scrivere il comando "python3 software.py" e lasciate andare)
